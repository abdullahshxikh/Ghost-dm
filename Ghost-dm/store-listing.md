## Full Description

**Maintain privacy in Instagram DMs while preserving your control over read receipts.**

Ghost DM is a lightweight Chrome extension that gives you control over Instagram read receipts. Read direct messages without letting senders know you've seen them, while still receiving read receipts from others when they read your messages.

### Key Benefits

• **One-Way Privacy**: Block outgoing "seen" notifications while keeping incoming receipts
• **Selective Invisibility**: Choose when to be invisible and when to show you've read messages  
• **Zero Impact**: Normal Instagram functionality remains completely intact
• **Instant Toggle**: Enable/disable Ghost Mode with a single click
• **Visual Feedback**: Clear indicators show when blocking is active

### Perfect For

• **Managing Expectations**: Avoid pressure to respond immediately to every message
• **Professional Boundaries**: Read work-related DMs without appearing always available
• **Personal Privacy**: Browse messages at your own pace without social obligations
• **Relationship Management**: Read sensitive conversations when you're ready to respond

### Privacy-First Design

• **100% Local Processing**: All functionality runs entirely in your browser
• **Zero Data Collection**: We never collect, store, or transmit any personal information
• **No External Servers**: No communication with third-party services
• **Open Source**: Fully transparent code available for review

### How It Works

Ghost DM uses Chrome's advanced declarativeNetRequest API to intercept specific Instagram read receipt requests before they reach Instagram's servers. This modern approach ensures:

• **High Performance**: No impact on browsing speed or Instagram functionality
• **Reliability**: Works consistently across Instagram updates
• **Security**: No access to message content or personal data
• **Efficiency**: Minimal resource usage and battery impact

### Easy to Use

1. Install the extension and click the Ghost DM icon in your toolbar
2. Toggle "Ghost Mode" ON when you want to read messages invisibly
3. Browse Instagram DMs normally - senders won't see "seen" notifications
4. Toggle OFF when you want to resume normal read receipts

### What You Get

• Modern dark-themed popup interface
• Real-time status indicators (green = active, white = inactive)
• Toast notifications when requests are blocked
• Keyboard shortcuts for quick access
• Responsive design that works on all screen sizes
